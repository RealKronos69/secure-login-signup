# secure-login-signup
**secure login &amp; signup made using tailwindcss, express, mongoose, bcrypt and jwt auth**

## Features
*   **Validation Check**
*   **Salted Password Hashing**: hashed password is stored using BCrypt .
*   **Token-Based Authentication**: Secure session handling using JWT (JSON Web Tokens).
*   **Error Handling**: Secure, generic error messages.
*   **Token Expiration**: user token is expired after a certain time.
*
-- will add more features later --

## Stack
* **Backend:** [Node.js,Express.js]
* **Database:** [MongoDB]
* **Authentication:** [JWt (jsonwebtoken)]
* **Styling:** [Tailwindcss]
